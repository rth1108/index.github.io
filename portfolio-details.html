Project information

  • 프로젝트 명
    AWS를 이용한 아키텍쳐 구성
  • 프로젝트기간 기간
    2022.12.11 ~ 2022.12.20
  • 프로젝트 목표
    DR(Disaster Recovery)에 개념을 공부하고 실제 구축을 통해 DR의 실시간 전송 및 복구 기능을 사용하고 DR 기능 중 하나인 Document DB에 서울 리전과 과 도쿄 리전을 연결하여 Global document cluster를 사용하였다. 웹사이트 배포와 동시에 방화벽 WAF를 사용하여 SQL injection과 XSS 방어를 목표로 하였다.
  • AWS 사용 기능
    EC2, S3, ALB, Auto scaling, WAF, Router 53, Document DB, BeansTalk
